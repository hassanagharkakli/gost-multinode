## Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

### v1.0.0 - Initial stable release

- First public, stable release of `gost-multinode`.
- Interactive `gost-manager` CLI for installing Gost and managing Iran/foreign nodes.
- Systemd units for one Iran relay node and multiple foreign nodes.
- Support for password and SSH key authentication between foreign nodes and the Iran relay.
- Basic security hardening, configuration validation, and English-only codebase.
